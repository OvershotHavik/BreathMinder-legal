# BreathMinder legal site setup

This folder contains a simple static website for your legal pages:
- `index.html`
- `privacy.html`
- `terms.html`
- `styles.css`

## Publish with GitHub Pages

1. Create a new **public** GitHub repository, for example: `breathminder-legal`
2. Upload all files from this folder into the root of that repo
3. In GitHub, open **Settings** → **Pages**
4. Under **Build and deployment**:
   - **Source:** Deploy from a branch
   - **Branch:** `main`
   - **Folder:** `/ (root)`
5. Save
6. Wait for GitHub Pages to publish
7. Your site URL will be shown in the Pages settings

## URLs to use

After publish, your links will usually be:
- `https://YOUR-USERNAME.github.io/REPO-NAME/`
- `https://YOUR-USERNAME.github.io/REPO-NAME/privacy.html`
- `https://YOUR-USERNAME.github.io/REPO-NAME/terms.html`

Use the `privacy.html` link in App Store Connect for the Privacy Policy URL.

## Optional next step

If you later buy a domain, you can connect it to GitHub Pages without rebuilding the site.
